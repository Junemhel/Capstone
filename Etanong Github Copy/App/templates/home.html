- An Electronic Learning Subject-Specific Assessment System for Basic Education Level Students

Aims to solve this problem by creating an environment where the student can interact with the teacher in a more fulfilling way, without having to call students individually during a discussion to ask them a question about the topic. e-Tanong emphasizes the following:

  • Teacher Auto Generate Question
  • Quiz Review
  • Quiz Statistics

e-Tanong Mo Na! Be Part!.


Features


Quiz Maker

Want to create quality, user- friendly quizzes in minutes? e-tanong provides everything you need, including analysis tools that simplify quiz scoring and help you with quiz reviewer.

Quiz Questions

With this application, you can choose what type of quizzes you want to put and present to your students.

Dashboard Display

Provide e-Tanong users with an intuitive, easy-to-read, easy-to-use tool for monitoring their datas.

Interactive Participation

e-Tanong takes this concept to the online classroom where each student is given a token to represent themselves which is visible to other students.

Quiz Statistics

Quiz summary showing statistics for all sections including the quiz average score, students who got the highest score sort for ranking, and average time of quiz completion.

True/False Question Type

A student is given only two choices for an answer in this kind of question: True or False. Students are given a token to represent themselves.
